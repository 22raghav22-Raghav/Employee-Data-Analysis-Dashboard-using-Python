Employee-Data-Analysis-Dashboard-using-Python
This project is a simple data analysis dashboard built using Python. It focuses on analyzing employee data and generating insights using basic data analysis and visualization techniques.
Project Overview

In this project, I created a dataset of employees and performed exploratory data analysis (EDA) to understand patterns related to gender distribution, departments, age, and salary trends. The project also generates a report automatically.

Technologies Used--
Python
pandas
matplotlib
seaborn
python-docx

Features--
Created and managed employee dataset
Performed data analysis using Pandas
Generated visualizations:
Gender distribution
Department distribution
Age distribution
Salary vs Experience
Automatically generated a Word report with graphs
Extracted basic insights from data

Key Insights--
Majority of employees are male
IT department has the highest number of employees
Most employees fall in the mid-age working group
Salary generally increases with experience

Output--
CSV file of employee data
Summary statistics file
Graphs (saved as images)
Final Word report (Employee_Report.docx)

Purpose:

The goal of this project is to practice basic data analysis, visualization, and reporting using Python. It is designed as a beginner-level project for learning and internship preparation.
